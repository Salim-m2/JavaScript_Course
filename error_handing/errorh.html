Error Handling in JS

Error is a clear indication that something went wrong.

In JavaScript error occurs because of different reasons. And because of this we different types of errors.

    Types of errors in JS

  • Parsing Error

    • Occurs due to wrong syntax.

  • Runtime Error

    • Occurs when the programm is running.

    Examples of errors in JS

  • ReferenceError

    • Happens when trying to access the variable that has not been defined/declared. Or in hoisting when trying to access the variable declared with let or const

  • TypeError

    • Occurs when trying to person a wrong operation on the wrong data type. For example doing value.toLowerCase() where value is a number.

  • RangeError

    • Thrown when the value is not within a set or a range.

  • SyntaxError

    • Thrown when the the code does not follow the rules of programming

  • URIError

    • Thrown during encoding or decoding.

  • EvalError
