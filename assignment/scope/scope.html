JavaScript Scope and Scope Chain

Scope: Is where the variable is accessed in JS

Scope Chain: Is the mechanism in JS that defines the hierachy of the scope allowing the machine to traverse from the current, inner most scope to the outermost scope

Types of scope

  • Global Scope

    Variables declared with var are the property of windows object in this scope. While those declared with let and const are not the property of the window object.

    The variables in this scope can be accessed anywhere in the file.

  • Function Scope

    Variables in this scope can only be accessed within this scope. Whether they are declared with var, let or const, they actually behave the same.

  • Block Scope

    Variables declared with let or const have a block scope while those declare with var have no block scope

  • Module Scope
