The this keyword in JS

The this keyword: Is like a friend or somebody who does not have a firm stand in an argument. In short the this keyword always changes its value depending on where it is defined.

Example of the different values of this in different lexical environment.

  • In the global scope

    The value of this is the window object

    console.log(this) // window

  • Inside an Object

    The value of this is the object in which the function is defined in.

  • Inside a function

    The value of this is bound to the windows object

  • Inside a nested function

    The value of this is bound to the windows object

  • Inside a function under strict mode

    The value of this is undefined.

  • Inside an arrow function

    An arrow function does not have its own this rather the value of this is the parent of the scope in which the arrow function is defined
